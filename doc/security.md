# Security features

1. It is required to change password every year. If you created/changed password one year ago, during log in process, you will be redirected to form where you have to set up new password.


2. Password must be at least 14 characters long.


3. We do not allow using same password that was used before. We remember up to 24 last passwords, so u can't use them another time.


5. We will suspend yours account, if there are 5 invalid sign in tries in a row. Account is suspended for 15 minutes, during which you can't log in and can't request password change. Also key personel is alerted about that situation.
